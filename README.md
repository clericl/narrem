# Narrem
A word processor and organization tool designed for writers of all shapes and colors. Built on Django with React. Work in progress!

## Overview
![fonts](https://media.giphy.com/media/QhWkFn3BO3FrG/giphy.gif)

Yeah, it happens. Narrem is here so you don't have to spend hours indenting each line of your screenplay perfectly.

Unless you want to. Sometimes you need that, you know?

## Features
Narrem is a work in progress and may have features not listed here (or not have features listed here, as its vision sharpens).
* Automatic manuscript formatting for plays, scripts, books, and more
* Character sheets and maps
* Notes and Events items for ideas and passages
* Storyboards to string disparate pieces together into timelines and full manuscripts
* Flash fiction practice: daily prompts with voting on submissions
* Peer editing: request reviews of your work from writers like you
